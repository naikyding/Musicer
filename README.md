# MUSICER MusicPlayer [:link:](http://220.128.133.15/s1080417/06_JS_OpenAPI/)
串接 youtube API，具有調整音量、播放、搜尋功能!
![](https://i.imgur.com/5Rvhkfe.jpg)

**開發工具/語言/技術：**
* Javascript ES6
* Jquery
* Html 5
* Bootstrap 4.4
* PHP 7
* Git
* AJAX 
* Sweetalert 2
* Ai

### 功能要點：
- 即時搜尋
- 可隱藏式播放器
- 播放跑馬燈
- 音量調整
- 自動熱門歌曲搜尋
- 無外部播放權限提示
